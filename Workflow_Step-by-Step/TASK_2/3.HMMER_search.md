### Step 3: Run HMMER search

**Aim**
To identify *Amborella trichopoda* MADS and K-domain proteins using Pfam-calibrated HMM profiles sourced from UniProt/InterPro.

---

### Procedure

1. **Navigate to Project Directory**:
   ```bash
   cd ~/Desktop/NIPGR-data/NIPGR_WORK/HMMER_project
   ```

2. **Execute HMMER Search**:
   ```bash
   # Perform hmmsearch for MADS and K-domain using Pfam profiles
   hmmsearch --tblout Amborella_trichopoda_MADS_hmmsearch.txt PF00319.MADS.hmm Amborella_trichopoda.fa
   hmmsearch --tblout Amborella_trichopoda_Kdomain_hmmsearch.txt PF01486.K_domain.hmm Amborella_trichopoda.fa
   ```
