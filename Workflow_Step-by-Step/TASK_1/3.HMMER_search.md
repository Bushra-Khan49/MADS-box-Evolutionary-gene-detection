# <h2 style="color:darkblue; font:Times New Roman;">HMMER Workflow — Step 03 (Arabidopsis → Rice example)
</h2> 
<br>

## **Aim**  
To shortlist candidates for Multiple Sequence Alignment (MSA).

## **Objective**
- Peform HMMER search of all the 3 domains of MADS-box in Arabidopsis i.e., PF00319.MADS, PF00847.AP2 AND PF01486.K_domain with our query sequence Oryza/rice.
- From results of HMMER search find common genes in MADS(PF00319) and K_domain(PF01486) and collect in single output file.
- Peform Blast search of the output file of intersecting results of MADS+K_domain and final blast results from self blast.

<br>

## Steps:-
```bash
#!/bin/bash
```
> ### Step 0: Installations
```
brew install hmmer seqkit blast
```
> #### <p align="centre">Version check</p>

```
hmmsearch -h
blastp -version
seqkit version
```
> ### Step 1: Directory and File check
```
cd ~/Desktop/NIPGR-data
ls -l

mkdir -p HMMER_project/{hmm,results,logs}

# Move HMMs into a tidy subfolder (optional but neat)
mv PF00319.MADS.hmm PF01486.K_domain.hmm PF00847.AP2.hmm Osativa_323_v7.0.protein.fa HMMER_project/hmm/

grep -c "^>" Osativa_323_v7.0.protein.fa   (gives the no. of ids present, should print 52424)
```
> ### Step 2: Peform HMM search and save results

```
cd ~/Desktop/NIPGR-data/HMMER_project

hmmsearch PF00319.MADS.hmm Osativa_323_v7.0.protein.fa > PF00319mads_hmmsearch_results.txt
hmmsearch PF01486.K_domain.hmm Osativa_323_v7.0.protein.fa > PF01486.K_domain_hmmsearch_results.txt
hmmsearch PF00847.AP2.hmm  Osativa_323_v7.0.protein.fa > PF00847.AP2_hmmsearch_results.txt

```
