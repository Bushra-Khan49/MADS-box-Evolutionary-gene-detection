# <h2 style="color:darkblue; font:Times New Roman;">Phase 3: Maximum Likelihood Phylogenetics — Step 01</h2> 
<br>

## **Aim**  
Convert the computationally heavy MAFFT alignments into mathematical evolutionary distance distances natively plotting ancestral relationships to *Arabidopsis* via Maximum Likelihood (ML) engines. 
<br>

## Steps:-

```bash
#!/bin/bash
```

> ### Step 0: Optimize Tree Builder Settings
You can either utilize the GUI of MEGA12 to build large trees, or use command-line equivalents like `megacc` or `IQ-TREE` to run the ML analysis automatically.
```bash
cd /Users/bushrakhan/Desktop/NIPGR-data/NIPGR_WORK
mkdir Phylogeny
cd Phylogeny

# A standardized substitution model parameter check (Example using IQ-TREE)
iqtree -s ../MSA/Clean_Aligned_AmbTr.fasta -m TEST -nt AUTO
```

> ### Step 1: Run The Maximum Likelihood Tree Builder
Here, we will compute 500 Bootstrap replicates to validate branch support.
```bash
# IQ-TREE automatically outputs distance files, standard .nwk newicks, and branch maps.
iqtree -s ../MSA/Clean_Aligned_AmbTr.fasta -m JTT+F+G4 -B 500 -T AUTO
```
*(If executing this using MEGA12 CC, you pass the `.meg` alignment along with `.mao` parameter files instead to produce the exact same standard Newick tree outputs).*

> ### Step 2: Extract Newick Format files
Retrieve the final `Amborella_trichopoda.nwk` formatted tree.
```bash
cp Clean_Aligned_AmbTr.fasta.treefile Amborella_trichopoda.nwk 
cat Amborella_trichopoda.nwk | head -n 1
```

*(This Newick file encodes the complex hierarchical branch placements of both species' MIKCc genes required for our visualization script next!)*
