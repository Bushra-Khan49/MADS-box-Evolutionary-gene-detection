# <h2 style="color:darkblue; font:Times New Roman;">MSA_search — Step 04 (Arabidopsis → Rice example)</h2>

---

## **Aim**  
To perform MSA and prepare a Phylogenetic tree.

## **Objective**
- Perform MSA using **MAFFT**.  
- Visualise MAFFT results using **AliView**.  
- Visualise Phylogenetic tree of the prepared output.  

---

## **Steps:-**

```bash
#!/bin/bash
```

>## Step 0: Installations
```
brew install hmmer seqkit blast

hmmsearch -h
seqkit version
```

>## Step 1: File check
```
pwd
ls -l
cd ..
pwd
ls -l
ls -d */
cd /Users/bushrakhan/Desktop/NIPGR-data/BLAST_project
ls -l
```

>## Step 2: Shortlist candidates (Extract intersection sequences)
```
seqkit grep -f MADS_Kdomain_intersection.txt/Combined_hmmersearch_results.txt Blast_final_shortlisted.fa > MSA_candidates.fa
```

>## Step 3: Combine files for MSA (From shortlisted candidates)
```
cat At_ABCDE_QUERY.fa MSA_candidates.fa > combined.fa
```

>## Step 3: Run MAFFT
```
mafft --auto combined.fa > MSA_resultsfile.fa
head MSA_resultsfile.fa
open -a TextEdit MSA_resultsfile.fa
```

>## Step 4: AliView Installation
```
brew --version
conda --version

conda activate base
conda config --add channels bioconda
conda config --add channels conda-forge
conda config --set channel_priority strict
conda install aliview
```

>## Step 5: Visualise with AliView 
```
aliview ~/Desktop/NIPGR-data/BLAST_project/MSA_resultsfile.fa
```
