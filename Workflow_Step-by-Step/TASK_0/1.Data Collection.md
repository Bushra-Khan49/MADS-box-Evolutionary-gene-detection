# Step 1. Data Collection

## Aim
To obtain high-quality genome, proteome, CDS, and gene annotation datasets for the model species and selected angiosperm lineages in order to perform comprehensive genome-wide identification and evolutionary analysis of MADS-box gene families.

---

## Tasks

- Identify all required sequence and annotation files for each species  
- Download complete proteomes for domain detection and orthology searches  
- Download genomes + GFF3 annotations for gene mapping and synteny analyses  
- Collect CDS/transcript sequences for validation and expression analyses  
- Retrieve Pfam HMM profiles for conserved domain identification  
- Record database source, version, release date, and storage size for reproducibility  

---

## Required Files per Species

| Data Type | Purpose | Format |
|-----------|------------------------------|----------------|
| Whole Genome | Genomic context, synteny | `.fa / .fna` |
| Proteome | Domain search, BLAST, MSA, phylogeny | `.fa / .faa` |
| CDS | Coding sequence validation | `.cds.fa` |
| Transcripts (optional) | Expression/RNA-seq mapping | `.cdna.fa / .mrna.fa` |
| Annotation | Gene coordinates/models | `.gff3` |
| Domains (Pfam) | HMM searches | `.hmm` |

---

## Databases Used

| Database | Purpose |
|-------------------------|----------------------------------------------|
| Phytozome (DOE JGI) | Most plant genome assemblies |
| TAIR | Arabidopsis reference & ABCDE genes |
| NCBI RefSeq/GenBank | Species not present in Phytozome |
| Pfam | Protein domain HMM profiles |

---

# 🌱 Species Dataset Summary

**Arabidopsis thaliana** was used as the model reference species due to its experimentally validated floral ABCDE genes, curated annotations, and well-characterized MADS-box clades, serving as the coordinate system for orthology and phylogenetic classification.

# ✅ Notes

- Always select the latest stable annotation release  
- Prefer primary transcript/protein sets to avoid redundancy  
- Record file size and version for reproducibility  
- Maintain organized directory structure:

```bash

  data/
   ├── genomes/
   ├── proteomes/
   ├── cds/
   ├── annotations/
   ├── domains/

```
