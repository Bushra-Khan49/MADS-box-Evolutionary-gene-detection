# Step 1. Data Collection

**Aim**  
To download Whole Genome, Whole Proteome, CDS, and annotations data for the model Rosaceae species *Prunus persica* (Peach) from Phytozome v2.1.

---

### Why *Prunus persica*?
*Prunus persica* (Peach) is the model species for the Rosaceae family. It possesses a relatively small diploid genome that hasn't undergone recent whole-genome polyploidization, unlike other Rosaceae members like strawberry. This makes it a high-quality reference for evolutionary studies on fruit development and wood formation in the Rosales order. The Phytozome v2.1 assembly provides a chromosome-level, well-annotated reference for genome-wide identification.

---

### Tasks
- Identify the *Prunus persica* v2.1 assembly on Phytozome (DOE JGI).
- Download the complete protein dataset for genome-wide MADS-box search.
- Retrieve the corresponding GFF3 and CDS files for gene model characterization.

### Databases Used
- **Phytozome (DOE JGI)** → Primary source for *Prunus persica* genomic data.
  - [https://phytozome-next.jgi.doe.gov](https://phytozome-next.jgi.doe.gov)
- **UniProt / InterPro** → Domain HMM profiles (Pfam: PF00319, PF01486).

### File Formats
| Data | File Type | Purpose |
|------|-----------|---------|
| Proteome | `Ppersica_298_v2.1.protein.fa` | Main search database |
| Genome | `Ppersica_298_v2.1.genome.fa` | Genomic mapping |
| Annotation | `Ppersica_298_v2.1.gene.gff3` | Site coordinates |
