# Step 1. Data Collection

**Aim**  
To download Whole Genome, Whole Proteome, CDS, and annotations data for the model legume *Medicago truncatula* (v4) from Phytozome.

---

### Why *Medicago truncatula*?
*Medicago truncatula* (Barrel Medic) is the established model for legume genomics and symbiosis research. It possesses a relatively small diploid genome compared to its polyploid relative, soybean, which makes it an ideal reference for determining the core legume MADS-box gene set. The Phytozome v4 assembly is high-quality and essential for resolving the phylogenetic relationships and gene structure of the Fabaceae-specific clades.

---

### Tasks
- Identify the *Medicago truncatula* v4 assembly on Phytozome (DOE JGI).
- Download the complete protein dataset for genome-wide MADS-box search.
- Retrieve the corresponding GFF3 and CDS files for gene model characterization.

### Databases Used
- **Phytozome (DOE JGI)** → Primary source for *Medicago truncatula* genomic data.
  - [https://phytozome-next.jgi.doe.gov](https://phytozome-next.jgi.doe.gov)
- **UniProt / InterPro** → Domain HMM profiles (Pfam: PF00319, PF01486).

### File Formats
| Data | File Type | Purpose |
|------|-----------|---------|
| Proteome | `Mtruncatula_198_v4.0.protein.fa` | Main search database |
| Genome | `Mtruncatula_198_v4.0.genome.fa` | Genomic mapping |
| Annotation | `Mtruncatula_198_v4.0.gene.gff3` | Site coordinates |
