# Step 1. Data Collection

**Aim**  
To download Whole Genome, Whole Proteome, CDS, and annotations data for the model monocot *Oryza sativa* (IRGSP-1.0 / v7.0) from Phytozome.

---

### Why *Oryza sativa*?
*Oryza sativa* (rice) is the primary model monocot and one of the world's most important cereal crops. As a monocot, it represents the major angiosperm clade that diverged from eudicots approximately 140–150 million years ago. Including rice in this study allows direct comparison of MADS-box clade sizes and subfunctionalization patterns between the monocot and eudicot lineages. The Phytozome v7.0 (IRGSP-1.0) assembly is the most comprehensive and widely used reference for rice genomics.

---

### Tasks
- Identify the correct assembly version (IRGSP-1.0 / v7.0) for *Oryza sativa*.
- Download the complete proteome file for genome-wide MADS-box identification.
- Retrieve the corresponding GFF3 annotation and CDS files from Phytozome.

### Databases Used
- **Phytozome (DOE JGI)** → Primary source for *Oryza sativa* v7.0 genomic data.
  - [https://phytozome-next.jgi.doe.gov](https://phytozome-next.jgi.doe.gov)
- **Pfam** → Domain HMM profiles (PF00319, PF01486).

### File Formats
| Data | File Type | Purpose |
|------|-----------|---------| 
| Proteome | `Osativa_323_v7.0.protein.fa` | Main search database |
| Genome | `Osativa_323_v7.0.genome.fa` | Genomic mapping |
| Annotation | `Osativa_323_v7.0.gene.gff3` | Site coordinates |

### Download Guide (Summary)
```bash
cd ~/Desktop/NIPGR-data/NIPGR_WORK
mkdir -p data/proteomes data/genomes data/annotations

# Download Oryza sativa v7.0 proteome from Phytozome
wget "https://phytozome-next.jgi.doe.gov/info/Osativa_v7_0" -O data/proteomes/Oryza_sativa.fa
```
