# <h2 style="color:darkblue; font:Times New Roman;">Phase 1: HMMER Profiling — Step 02</h2> 
<br>

## **Aim**  
Scan the bulk proteome data to conditionally extract only sequences containing the highly conserved MADS-box DNA binding domain utilizing Hidden Markov Models (HMMER) and the established SRF-TF (PF00319) Pfam profile.
<br>

## Steps:-

```bash
#!/bin/bash
```

> ### Step 0: Download the MADS-box Pfam Profile
First, retrieve the PF00319 Hidden Markov Model seed from the InterPro/Pfam database.
```bash
cd /Users/bushrakhan/Desktop/NIPGR-data/NIPGR_WORK
mkdir HMM_Profiling
cd HMM_Profiling

wget "https://www.ebi.ac.uk/interpro/api/entry/pfam/PF00319/?annotation=hmm" -O SRF-TF_PF00319.hmm
```

> ### Step 1: Execute `hmmsearch` against Species Proteomes
Use the HMM profile to search the target species proteomes. Set a strict E-value threshold (`1e-5`) to avoid false positives.
```bash
# Example for Amborella trichopoda
hmmsearch --cut_ga --domE 1e-5 -E 1e-5 \
    -o AmbTr_HMM_output.txt \
    --tblout AmbTr_hits.tbl \
    SRF-TF_PF00319.hmm \
    ../Raw_Proteomes/AmbTr.fa
```
*(Repeat this command across all 9 target species replacing the input and output filenames)*

> ### Step 2: Extract Hit Sequence IDs & Remove Duplicates
Extract just the raw Gene IDs from the HMMER tabular output to cleanly prepare for sequence pulling. It is critical to explicitly enforce duplicate removal here so that redundant IDs are collapsed.
```bash
# The 'sort | uniq' pipeline ensures any multi-domain duplicate hits on the same gene are cleanly merged.
grep -v "^#" AmbTr_hits.tbl | awk '{print $1}' | sort | uniq > AmbTr_MADS_IDs.txt
cat AmbTr_MADS_IDs.txt | wc -l
# Output: [Count of putative MADS-box genes]
```

> ### Step 3: Pull FASTA Sequences for Putative Hits
Use `seqtk` or a custom python script to extract the full protein sequences of our filtered IDs from the massive raw FASTA.
```bash
# seqtk subseq [in.fasta] [id_list.txt] > [out.fasta]
seqtk subseq ../Raw_Proteomes/AmbTr.fa AmbTr_MADS_IDs.txt > AmbTr_putative_MADS.fa
```
