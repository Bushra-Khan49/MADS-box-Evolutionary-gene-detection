# <h2 style="color:darkblue; font:Times New Roman;">Phase 3: Automated Subclade Annotation — Step 02</h2> 
<br>

## **Aim**  
Bridge the Phase 2 BLAST predictions with established *Arabidopsis* literature to definitively map 100% of our unknown species' hits to their biological functional clades programmatically.
<br>

## Steps:-

```bash
#!/bin/bash
```

> ### Step 0: The Functional Anchor Dictionary
We created a custom Python script [`build_annotation_table.py`](../../scripts/build_annotation_table.py) located centrally in the `/scripts` directory. Inside this script resides a hard-coded mapping dictionary connecting known AT identifiers strictly back to their functional biological classifications (e.g. `AT1G69120` ➔ `AP1/FUL/CAL`).

> ### Step 1: Execute Python Application against Datasets
Navigate to your project directory.
```bash
cd /Users/bushrakhan/Desktop/NIPGR-data/NIPGR_WORK
ls -l scripts/build_annotation_table.py
```
This application recursively reads the specific Species-vs-Arabidopsis BLAST files (`TopHits_AT.txt`) from each isolated species folder. Based on the target species ID, it matches the *Arabidopsis* identifier to the `AT_TO_CLADE` dictionary.
```bash
python3 scripts/build_annotation_table.py
```

> ### Step 2: Validate the Output MIKCc Annotation Table
The script outputs a finalized TSV database mapping the species genes to their functional clades.
```bash
cat MIKCc_Subclade_Annotation_Table.tsv | head -n 10
# Species   Gene_ID                TopHit_AT       %Identity   Subclade
# AmbTr     evm.model_27...        AT5G20240       52%         AP3/PI
# ...
```

*(This TSV holds the absolute predictive profiles for our thousands of genes and dictates their explicit coloring behavior in Phase 4!)*
