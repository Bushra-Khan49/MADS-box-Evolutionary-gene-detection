# Step 1. Data Collection

**Aim**  
To download Whole Genome, Whole Proteome, CDS, and annotations data for the model species *Arabidopsis thaliana* (TAIR10).

---

### Tasks
- Obtain the reference MADS-box protein sequences (ABCDE genes) for Arabidopsis.
- Download the complete proteome (TAIR10) for domain detection and identification.
- Retrieve the TAIR10 GFF3 annotation file for genomic mapping.

### Databases Used
- **TAIR (The Arabidopsis Information Resource)** → Primary source for Arabidopsis genomic data.
  - [https://www.arabidopsis.org](https://www.arabidopsis.org)
- **Pfam** → Protein domain HMM profiles (MADS: PF00319, K-box: PF01486).
  - [https://pfam.xfam.org](https://pfam.xfam.org)

### File Formats
| Data | File Type | Purpose |
|------|-----------|---------| 
| Proteome | `TAIR10_pep_20101214` | Primary search space |
| Genome | `TAIR10_genome_release` | Mapping and synteny |
| Annotation | `TAIR10_GFF3_genes` | Structural coordinates |
| Domains | `.hmm` | Domain identification |

### Download Guide (Summary)
```bash
cd ~/Desktop/NIPGR-data/NIPGR_WORK
mkdir -p data/proteomes data/genomes data/annotations data/domains

# Download TAIR10 proteins
wget "https://www.arabidopsis.org/download_files/Proteins/TAIR10_pep_20101214" -O data/proteomes/Arabidopsis_thaliana.fa
```
