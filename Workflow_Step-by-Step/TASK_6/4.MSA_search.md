Step 4: Multiple Sequence Alignment (MSA)

Aim To align identified candidate MADS-box protein sequences using MAFFT, MUSCLE, and Clustal Omega for comparative phylogenetic analysis.

Procedure

Navigate to Project Directory:
cd ~/Desktop/NIPGR-data/HMMER_project

Combine files for MSA:
# Merge query and candidates into one file for alignment
cat At_ABCDE_QUERY.fa Shortlisted_candidates.txt > {species_fa}_combined_for_msa.fa

Run Alignments:
# 1. MAFFT (Primary)
mafft --auto {species_fa}_combined_for_msa.fa > {species_fa}_MAFFT_Results.fa

# 2. MUSCLE (Secondary)
muscle -in {species_fa}_combined_for_msa.fa -out {species_fa}_MUSCLE_Results.fa

# 3. Clustal Omega (Tertiary)
clustalo -i {species_fa}_combined_for_msa.fa -o {species_fa}_ClustalO_Results.fa
