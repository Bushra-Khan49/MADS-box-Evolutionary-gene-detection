> ### Step 4: Multiple Sequence Alignment (MSA)

**Aim**  
To align identified *Nelumbo nucifera* candidate MADS-box protein sequences using MAFFT and perform visual inspection for conserved motifs.

---

> ### Step 4.1: Combine files for MSA
```bash
cd ~/Desktop/NIPGR-data/NIPGR_WORK/HMMER_project

# Merge query and candidates into one file for alignment
cat At_ABCDE_QUERY.fa Lotus_Shortlisted_candidates.fa > Lotus_msa_input.fa
```
[Combined file](https://github.com/Bushra-Khan49/MADS-box-Evolutionary-gene-detection/tree/main/Workflow_Step-by-Step/TASK_10/uploads)

---

> ### Step 4.2: Run MAFFT Alignment
```bash
cd ~/Desktop/NIPGR-data/NIPGR_WORK/HMMER_project

# Perform multiple sequence alignment
mafft --auto Lotus_msa_input.fa > Lotus_MSA_results.fa

# Preview the alignment
head Lotus_MSA_results.fa
open -a TextEdit Lotus_MSA_results.fa
```
[MSA Results](https://github.com/Bushra-Khan49/MADS-box-Evolutionary-gene-detection/tree/main/Workflow_Step-by-Step/TASK_10/uploads)
