### Step 3: Run HMMER search

**Aim**
To identify *Cinnamomum kanehirae* MADS and K-domain proteins using Pfam-calibrated HMM profiles sourced from UniProt/InterPro.

---

### Procedure

1. **Navigate to Project Directory**:
   ```bash
   cd ~/Desktop/NIPGR-data/NIPGR_WORK/HMMER_project
   ```

2. **Execute HMMER Search**:
   ```bash
   # Perform hmmsearch for MADS and K-domain using Pfam profiles
   hmmsearch --tblout Cinnamomum_kanehirae_MADS_hmmsearch.txt PF00319.MADS.hmm Cinnamomum_kanehirae.fa
   hmmsearch --tblout Cinnamomum_kanehirae_Kdomain_hmmsearch.txt PF01486.K_domain.hmm Cinnamomum_kanehirae.fa
   ```
