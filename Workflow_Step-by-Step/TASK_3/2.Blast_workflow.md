# Step 2: BLAST workflow — (*Nymphaea colorata*)

## Aim
To identify potential MADS-box genes by performing a BLASTP search against the *Nymphaea colorata* proteome using verified *Arabidopsis thaliana* reference sequences under stringent E-value and tabular reporting standards.

---

## Tasks
- Construct a BLAST database for the *Nymphaea colorata* v2.0 proteome.
- Execute BLASTP with e-value 1e-5 and outfmt 7 (Tabular with comment lines).
- Shortlist high-confidence candidates for domain validation.

---

## Command-Line Execution
```bash
cd ~/Desktop/NIPGR-data/NIPGR_WORK/BLAST_project

# Create a BLAST database for Nymphaea colorata
makeblastdb -in Nymphaea_colorata.faa -dbtype prot -out nymphaea_db

# Perform Blastp search with E-value 1e-5 and outfmt 7
blastp -query Arabidopsis_ABCDE_QUERY.fa -db nymphaea_db -out Nymphaea_colorata_Blast_results.txt -evalue 1e-5 -outfmt 7
```
