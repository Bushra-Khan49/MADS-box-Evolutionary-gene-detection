# <h2 style="color:darkblue; font:Times New Roman;">Phase 1: Data Acquisition — Step 01</h2> 
<br>

## **Aim**  
Download and prepare the complete raw protein (proteome) sequences for all 9 target species and the *Arabidopsis thaliana* reference dataset from major genomic repositories (Phytozome, Ensembl, NCBI).
<br>

## Steps:-

```bash
#!/bin/bash
```

> ### Step 0: Navigate to local project directory
```bash
cd /Users/bushrakhan/Desktop/NIPGR-data/NIPGR_WORK
mkdir Raw_Proteomes
cd Raw_Proteomes
pwd
```

> ### Step 1: Download Target Species Data
*(Note: Downloading instructions vary based on platform. Here is an example of fetching FASTA data using generalized `wget` links or local transfers.)*
```bash
# Example retrieval commands
wget "http://path.to.genomic.database/Amborella_trichopoda.protein.fasta" -O AmbTr.fa
wget "http://path.to.genomic.database/Glycine_max.protein.fasta" -O GlyMax.fa
wget "http://path.to.genomic.database/Oryza_sativa.protein.fasta" -O OrySat.fa
# ... Repeat for all 9 species
```

> ### Step 2: Download Arabidopsis Reference Data
*Arabidopsis thaliana* is utilized as the evolutionary framework to annotate our target subsets later.
```bash
wget "https://www.arabidopsis.org/download_files/Proteins/TAIR10_pep_20101214" -O Arabidopsis_TAIR10_proteins.fa
```

> ### Step 3: Verify Folder Contents and Unzip Files
```bash
# Extract any compressed archives
gunzip *.gz
ls -lh
```
