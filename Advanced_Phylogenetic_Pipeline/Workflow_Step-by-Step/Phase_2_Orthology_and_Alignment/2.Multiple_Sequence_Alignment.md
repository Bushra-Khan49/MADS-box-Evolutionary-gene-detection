# <h2 style="color:darkblue; font:Times New Roman;">Phase 2: Multiple Sequence Alignment (MSA) — Step 02</h2> 
<br>

## **Aim**  
Perform mathematically precise alignments scaling both sequence divergence globally across *Arabidopsis* and target species locally. The alignment specifically relies on M and K domain conservation to generate proper structural overlap while computing sequence gap logic strictly through MAFFT or MUSCLE.  

We prepare a single `.fa` (FASTA) file consolidating ALL *Arabidopsis* reference genes combined natively with the selected specific target species *Amborella trichopoda*.
<br>

## Steps:-

```bash
#!/bin/bash
```

> ### Step 0: File Consolidation
It is critical that the alignments occur *co-dependently*. To force the ML algorithm to trace true evolutionary distance, the *Arabidopsis* and the target species *must* be aligned within the same exact file.
```bash
cd /Users/bushrakhan/Desktop/NIPGR-data/NIPGR_WORK
mkdir MSA
cd MSA

# Concatenate Amborella genes with Arabidopsis references
cat ../Domain_Validation/1.AmbTr_MIKCc_Final.fasta ../Arabidopsis_BLAST/AT_ALL_MADSbox_PROTEINseq.txt.fa > Combined_AmbTr_AT.fasta
```

> ### Step 1: Execute Parameterized `MAFFT/MUSCLE` Alignment
We apply algorithms to ensure precision around the highly-conserved 'M' (MADS) and 'K' (Keratin-like) domains. We will permit generous gaps in the inherently variable 'I' and 'C' regions.
```bash
# Example showing an extended MAFFT run for the combined file:
mafft --maxiterate 1000 --localpair Combined_AmbTr_AT.fasta > Aligned_AmbTr_AT.fasta
```
*(If building a 600-gene Master Tree across all 9 species, simply `cat` all 9 final fasta files into one before running `mafft`!)*

> ### Step 2: Quality Trimming & Processing (Optional / Automatic via MEGA)
The raw output `.fasta` often needs to be trimmed of poorly aligned end regions. Tools like `trimAl` can be used. Otherwise, the alignment is natively parsed into a file format appropriate for IQ-TREE / MEGA.
```bash
trimal -in Aligned_AmbTr_AT.fasta -out Clean_Aligned_AmbTr.fasta -gappyout
```

*(This resulting MSA is now mathematically ready to be fed into an ML tree builder!)*
