### Step 4: Multiple Sequence Alignment (MSA)

**Aim**
To align identified *Medicago truncatula* candidate MADS-box protein sequences using MAFFT and perform visual inspection for conserved motifs.

---

### Procedure

1. **Combine files for MSA**:
   ```bash
   cd ~/Desktop/NIPGR-data/NIPGR_WORK/HMMER_project

   # Merge query and candidates into one file for alignment
   cat At_ABCDE_QUERY.fa Medicago_Shortlisted_candidates.fa > Medicago_msa_input.fa
   ```
[Combined file](https://github.com/Bushra-Khan49/MADS-box-Evolutionary-gene-detection/tree/main/Workflow_Step-by-Step/TASK_7/uploads)

2. **Run MAFFT Alignment**:
   ```bash
   cd ~/Desktop/NIPGR-data/NIPGR_WORK/HMMER_project

   # Perform multiple sequence alignment
   mafft --auto Medicago_msa_input.fa > Medicago_MSA_results.fa

   # Preview the alignment
   head Medicago_MSA_results.fa
   open -a TextEdit Medicago_MSA_results.fa
   ```
[MSA Results](https://github.com/Bushra-Khan49/MADS-box-Evolutionary-gene-detection/tree/main/Workflow_Step-by-Step/TASK_7/uploads)

3. **AliView Installation**:
   If AliView is not installed, follow these steps:
   ```bash
   # Option 1: macOS via Homebrew
   brew install --cask aliview

   # Option 2: via Conda (Bioconda)
   conda activate base
   conda config --add channels bioconda
   conda config --add channels conda-forge
   conda install aliview
   ```

4. **Visualize Alignment**:
   ```bash
   aliview ~/Desktop/NIPGR-data/NIPGR_WORK/HMMER_project/Medicago_MSA_results.fa
   ```
