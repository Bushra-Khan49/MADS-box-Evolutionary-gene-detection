### Step 2: BLAST workflow

**Aim**
To identify homologous MADS-box genes using reference Arabidopsis ABCDE organ identity queries.

---

### Procedure


```bash
cd ~/Desktop/NIPGR-data/NIPGR_WORK/BLAST_project

# Create a BLAST database for Helianthus annuus proteome
makeblastdb -in Helianthus_annuus.fa -dbtype prot -out sun_db

# Perform Blastp search using Arabidopsis ABCDE reference genes
blastp -query At_ABCDE_QUERY.fa -db sun_db -out Helianthus_annuus_Blast_results.txt -evalue 1e-5 -outfmt 7
```
